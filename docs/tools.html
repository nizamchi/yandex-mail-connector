yandex_unanswered Без ответа (неотвеченные) L0 ✓ Список полученных тредов, где последнее письмо — от другого человека и старше N дней. Работает по локальному индексу, группирует треды между папками. Результаты ранжируются и помечаются (tier + reasons): треды с флажком и переписки с вашим участием — наверху; авто-уведомления — внизу, но не скрываются (recall=100%).

Конфиг: YANDEX_ATTACHMENT_DIR (default: временный каталог ОС) / YANDEX_ATTACHMENT_MAX_BYTES (default: 25 МиБ).
yandex_find_attachments Поиск вложений (каталог) L0 ✓ Поиск вложений по локальному каталогу (метаданные BODYSTRUCTURE), без скачивания писем и без IMAP. Фильтры from / filename_contains / mime / since / before (подстрока). Одинаковые файлы (имя + размер) схлопываются с occurrence_count и списком мест (folder#uid). Каталог считает вложением и встроенные именованные части (логотипы) — отсекай точным mime (напр. application/pdf). Ship-критерий слоя: find_attachments(from, mime='application/pdf') < 50 мс.
